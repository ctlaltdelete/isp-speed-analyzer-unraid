# ISP Speed Analyzer for Unraid
Streamlit dashboard for automatic Ookla Speedtests with logging, alerts, and daily average graphs.

### Quick Start
1. Place this folder in `/mnt/user/docker/isp-analyzer/`
2. Build and run:
   \`\`\`bash
   docker compose up -d
   \`\`\`
3. Open [http://your-unraid-ip:8501](http://your-unraid-ip:8501)
